LeagueAutoAccept will automatically accept all match found requests as soon as the queue pops. It communicates with the League client via its internal API.

It is currently tested for Python 3.12.

Please see the [Justfile](https://github.com/casey/just) for running this program.

Optional Development Tools:
- [just](https://github.com/casey/just)
- [Python uv](https://github.com/astral-sh/uv)
